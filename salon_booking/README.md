# 💇‍♂️ Salon Booking App (Flutter – UI Only)

A **Flutter-based Salon Booking Application** built using **GetX architecture**.  
This repository contains **frontend (UI + state management) only**.  
⚠️ **Firebase has been completely removed** from this project.

This version is intended to be integrated with a **custom backend (Node / Django / Spring / etc.)**.

---

## 🚀 Features

### 👤 User Panel
- Login & Register UI
- Salon browsing UI
- Service selection
- Booking form UI
- User home screen

### 🛠 Admin Panel
- Admin dashboard UI
- Manage bookings UI
- Manage staff UI
- Manage services UI
- Manage offers UI
- Inventory UI
- Gallery UI
- Reviews UI
- Profile & settings UI

> ⚠️ All data is currently **mocked / in-memory** using GetX controllers.

---

## 🧱 Tech Stack

- **Flutter**
- **Dart**
- **GetX**
  - State Management
  - Routing
  - Dependency Injection
- Clean MVC-style folder structure
